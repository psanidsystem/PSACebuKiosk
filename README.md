# PSA Cebu Kiosk Starter Project

This is a beginner-friendly Android Studio starter project for a PSA Cebu kiosk app built with Kotlin and Jetpack Compose.

## Included screens
- Home
- Civil Registry
- National ID
- Citizen's Charter
- Public Bulletin
- Office Guide
- Statistics & Info

## Open in Android Studio
1. Open Android Studio.
2. Click **Open**.
3. Select the `PSACebuKiosk` folder.
4. Let Android Studio sync the project.
5. If Android Studio asks to install the Android SDK or update Gradle settings, accept the recommended setup.
6. Run the app on an emulator or Android device.

## Notes
- This project is a starter UI only.
- The National ID buttons are placeholders for your appointment system integration.
- Content is sample text so you can learn step by step.

## Recommended next step
After you open the project successfully, build the Civil Registry details screen first.
